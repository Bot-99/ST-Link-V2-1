# STLINK-V2.1
## HKUST Robotics Team
## Func
TypeC + STlink + TTL

## cfg
stlink-v2.cfg / stlink-v2_1.cfg

## How to setup
1. connect the swd to other stlink
2. download the firmware
  - open cubeprogrammer / stlink utility
  - connect
  - open the firmware
  - write file (firmware)  
3. desoldering the 0R resistor (R19 R20)
